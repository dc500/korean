Exceptions

Rule 1

ㄷ ⇨ ㄹ

듣다 ⇨ 들…

닫다 ⇨ 닫아요

받다 ⇨ 받아요

믿다 ⇨ 믿어요

ㅂ ⇨ 우

줍다 ⇨ 주워…

돕다 ⇨ 도와요

입다 ⇨ 입어요

좁다 ⇨ 좁아요

잡다 ⇨ 잡아요

Rule 2

Same as exception rule 1, and:

ㄹ ⇨ delete, but use as if vowel, not consonant

살다 ⇨ 삽시다

Rule 3

Same as exception rule 1, and:

ㄹ ⇨ ㄹ, but use as if vowel, not consonant

살다 ⇨ 살면
